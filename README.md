# Can I Search

You Can Search

## Features
search something

* web: shift + alt + w

* npm: shift + alt + n

* translate: shift + alt + t
## Extension Settings
You can configure different parameters, for example
```
"can-i-search":{
    "web":"https://www.baidu.com/s?ie=utf-8&f=8&wd={query}",
    "npm":"",
    "translate":""
}
```
